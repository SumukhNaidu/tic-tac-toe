# 🌌 Cosmic Tic-Tac-Toe

A stunning, interactive Tic-Tac-Toe game with a cosmic-themed UI and smooth animations!

## 🎮 Features

- **Beautiful Cosmic Design** - Animated gradient background with a space theme
- **Interactive Gameplay** - Play against another player locally
- **Score Tracking** - Keep track of wins, draws, and losses
- **Responsive Design** - Works on desktop and mobile devices
- **Smooth Animations** - Glowing text effects and gradient overlays
- **Reset & New Game Options** - Start fresh or begin a new round anytime

## 🎯 How to Play

1. Open `tic tac toe.html` in your web browser
2. The game board is a 3x3 grid
3. Players take turns clicking on empty squares
4. First player is **X**, second player is **O**
5. Win by placing 3 of your marks in a row (horizontal, vertical, or diagonal)
6. Click **"New Game"** to start a new round
7. Click **"Reset Game"** to clear all scores and start over

## 🚀 Getting Started

### Option 1: Direct File
1. Download or clone the repository
2. Open `tic tac toe.html` in any modern web browser
3. Start playing!

### Option 2: Local Server (Recommended)
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if http-server is installed)
http-server
```

Then visit `http://localhost:8000` in your browser.
